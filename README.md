# Antragsformular-fuer-Studierende

Dieses Repository enthält ein interaktives Web-Formular zur struktuierten Erfassung und Verwaltung von Studierendendaten. Es dient als Interface, um persönliche Details und IT-Spezialisierungen effizient zu dokumentieren.

## Projektbeschreibung
Das Formular ermöglicht die digitale Aufname von Studenten und deren Zuweisung zu spezifischen Ausbildungspfaden. Es wurde entwicklet, um eine saubere Datenstruktur für die Verwaltung von IT-Kursen zu gewährleiten.

##Features & erfasste Daten.
Das Formular deckt folgende Informationen ab:

* **Persönliche Details:** Vorname, Nachname und Geschlechtsauswahl.
* **Trainings-Module:** Auswahl spezialisierter Fachrichtungen:
* Webentwicklung (HTML5, CSS3, JavaScript)
* Java Fundamentals
* C# Fundamentals
* CCNA (Cisco Certified Network Associate)
* Python & Data Engineering

## Technologien
* **HTML5:** Struktur des Formulars und semantische Elemente.
* **CSS3:** Modernes Design und Responsive Layot für mobile Endgeräte.
* **GitHub Pages:** Live-Hosting der Anwendung.
